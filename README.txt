Zestark Visual Timer v1.1

〇 概要
シンプルなビジュアルタイマーアプリです。
〇 動作環境
・Windows 10 / 11

〇 使い方
1. ZestarkTimer.exe を起動
2. 時間を設定
3. スタートボタンで開始

〇 ファイル構成
・ZestarkTimer.exe : アプリ本体
・readme.txt : このファイル

〇 注意事項
・初回起動時にWindowsの警告が出る場合があります（自己責任で許可してください）
・ウイルスではありません（自己署名または未署名のため警告が出ることがあります）
