# ビジュアルバータイマーアプリ 要件定義書

## 1. システム概要

本アプリケーションは、時間の経過を視覚的に把握できるビジュアルバータイマーを提供する。作業・学習・会議などの時間管理を、直感的かつ正確に行えるツールとして利用する。

主な機能は以下の2つで構成される。

- 時計機能
- ビジュアルバータイマー機能

---

## 2. 動作環境

| 項目 | 内容 |
|---|---|
| 対応OS | Windows / macOS / Linux |
| ネットワーク | オフライン動作可（インターネット接続不要） |

---

## 3. 機能要件

### 3.1 時計機能

現在時刻とタイムゾーンを表示する。

| 項目 | 内容 |
|---|---|
| 表示内容 | 現在時刻・タイムゾーン名 |
| 表示形式 | `HH:MM:SS` + タイムゾーン名（例：Tokyo） |
| タイムゾーン設定 | ユーザーが任意のタイムゾーンを選択可能 |

### 3.2 ビジュアルバータイマー機能

#### タイマー設定

| 項目 | 内容 |
|---|---|
| 設定単位 | 分単位のみ（秒単位での設定は不可） |
| 最小時間 | 1分 |
| 最大時間 | 60分 |

#### ビジュアルバー

| 項目 | 仕様 |
|---|---|
| メモリ数 | 30個（固定） |
| 表示方向 | 横方向 |
| 1メモリの時間 | 設定時間 ÷ 30 |
| 表示目的 | 経過時間を視覚的に把握するため |

時間の経過に応じて右からメモリが点灯し、00分00秒で全点灯する。

#### 残り時間表示

秒単位で正確に表示する。

```
MM分SS秒（例：29分59秒）
```

#### 表示レイアウト

ビジュアルバーと残り時間は横並びで表示する。

```
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■  30分00秒
```

### 3.3 操作機能

| 操作 | 説明 |
|---|---|
| 開始 | タイマーを開始する |
| 停止 | タイマーを一時停止する |
| 再開 | 停止中のタイマーを再開する |
| リセット | タイマーを初期状態に戻す |

### 3.4 通知・アラーム機能

タイマーが `00分00秒` になった時点で以下を実行する。

- 通知を画面に表示する
- アラーム音を再生する
- ユーザーが停止操作を行うまで音を継続する

---

#### 4. 画面仕様

メイン画面の構成は以下のとおり。デスクトップの隅に配置しやすいよう、コンパクトな固定サイズ（例：320px × 360px）で構成する。

```
+-------------------------------+
```
|        14:32:10               |
|          Tokyo                |
|                               |
| ■■■■■■■■■■■■■■■■■  30分00秒   |
|                               |
|   [開始]  [停止]  [リセット]  |
+-------------------------------+
```

---

## 5. 非機能要件

| 項目 | 要件 |
|---|---|
| タイマー誤差 | ±1秒以内 |
| パフォーマンス | 低CPU使用率で動作すること |
| 安定性 | 長時間動作でもクラッシュしないこと |
| 操作性 | 直感的に操作・把握できるシンプルなUI |
| ウィンドウ設定 | 320x360px程度の固定サイズ。リサイズ不可。 |
